# 🛒 Modern Ecommerce Website

An elegant, full-stack **Ecommerce Web Application** built using **React + Vite** on the frontend and **Spring Boot** on the backend, offering seamless shopping experiences with modern performance and security standards.

---

## 🚀 Key Features

- 🏠 Responsive homepage with featured products
- 🛍️ Product listings with dynamic data
- 🧺 Shopping cart functionality
- 🔄 Client-side routing with `react-router-dom`
- 🌐 REST API integration with Spring Boot
- 📦 Backend handles products, orders, and users
- 🎨 Bootstrap 5 and Sass for responsive design
- ⚡ Vite for fast development and HMR
- 📡 Axios for API calls

---

## 🛠️ Tech Stack

### 🎯 Frontend
- **React 18**
- **Vite**
- **React Router DOM**
- **Axios**
- **Bootstrap 5** + **Sass**
- **React Bootstrap** and **React Icons**

### 🚀 Backend
- **Spring Boot**
- **Java 21**
- **Spring Data JPA**
- **RESTful APIs**
